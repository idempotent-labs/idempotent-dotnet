Idempotent.Redis provides a distributed cache-backed idempotent store using Redis. It enables idempotent behavior across multiple application instances.

Redis is used as a coordination and caching layer, not a relational store.

ðŸš€ Quickstart

Install Package

dotnet add package Idempotent.Redis

Register Store

builder.Services.AddRedisStore(
    storeKey: "default",
    clientName: "AspNetCoreApplication",
    endPoint: "localhost:6379",
    allowAdmin: true,
    abortOnConnectFail: false,  // Optional, default value is false
    connectRetry: 5,            // Optional, default value is 5
    connectTimeout: 10_000,     // Optional, default value is 10_000
    syncTimeout: 5_000          // Optional, default value is 5_000
);
