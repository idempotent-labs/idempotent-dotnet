üåê Idempotent.AspNetCore

Idempotent.AspNetCore integrates the Idempotent.Core with ASP.NET Core using middleware, endpoint metadata, and dependency injection. This package translates HTTP requests into idempotent transactions and replays responses when applicable.

The store implementation is resolved using keyed DI.

üß¨ Characteristics

  • Intercept HTTP requests using middleware
  • Capture request and response bodies safely
  • Enforce idempotent behavior at runtime
  • Replay completed responses
  • Integrate with ASP.NET Core routing and DI

üöÄ Quickstart

Install Package

dotnet add package Idempotent.AspNetCore

Setup Store (at least one)

Idempotent.InMemory Idempotent.Redis Idempotent.SqlServer Idempotent.DynamoDb

Register Services

builder.Services.AddIdempotentServices(
    storeKey: "default",
    headerKey: "Idempotent-Key",
    bufferThresholdBytes: 32_768        // Threshold Upper Bound: 32 x 1_024 = 32_768 (32 KB)
);

Register Middleware

builder.BuildIdempotentMiddleware(
    methods: [ "POST", "PATCH" ],
    headerKey: "Idempotent-Key",
    forceContinueWithoutIdempotent: false,
    bufferThresholdBytes: 32_768,       // Threshold Upper Bound: 32 x 1_024 = 32_768 (32 KB)
    requestBodyLimitBytes: 524_288,     // Limit Upper Bound: 512 x 1_024 = 524_288 (512 KB)
    responseBodyLimitBytes: 1_048_576,  // Limit Upper Bound: 1_024 x 1_024 = 1_048_576 (1 MB)
    appendTruncatedBodyInResponse: true
);
